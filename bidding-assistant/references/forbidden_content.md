# 投标文件禁止内容清单

## 绝对禁止：中标概率和匹配度自评

以下内容**绝对不能**出现在正式投标文件中：

| 禁止内容 | 原因 | 正确处理 |
|----------|------|---------|
| "我方中标概率 XX%" | 不专业、不合规，等于自曝底牌 | 仅作为内部决策参考，另存为独立分析文件 |
| "综合匹配度 XX分" | 自说自话，评标专家不认可 | 删除，改为客观的功能/条款对照表 |
| "XX维度评分 XX分" | 主观自评，无公信力 | 删除，用事实和数据说话 |
| "技术能力匹配度 93%" | 自我打分毫无意义 | 改为客观描述产品功能+对应招标条款 |
| "经评估，我方在此项目上具有明显优势" | 过度自信表述 | 改为"我方产品功能完整覆盖招标要求" |
| "中标概率预估：80%" | 评估结论写入投标文件极不专业 | 移到内部参考文件 |

## 应避免的表述

- 任何以"内部评估""我方自评""预估"开头的段落
- 包含雷达图、评分卡、柱状图等自评可视化的内容
- "相比竞争对手，我方具有..." 的比较性表述
- "毫无疑问地""绝对地""100%满足"等过度绝对化用词

## 正确的替代方式

| 不要写 | 应该写 |
|--------|--------|
| "我方中标概率80%" | 删除 |
| "技术匹配度93分，为所有投标人中最高" | "我方产品覆盖招标文件要求的全部7大功能模块，详见第四章功能对照表" |
| "经AI评估，我方综合得分最高" | "以下逐条展示我方对招标文件各项要求的响应方案" |

## 内部参考 vs 正式投标文件

**内部参考（可以包含评估内容）：**
- 匹配度分析报告
- 中标概率预估
- SWOT分析
- 竞争对手分析
- 风险评估矩阵

**正式投标文件（只允许客观内容）：**
- 功能对照表
- 资质证书
- 技术方案
- 案例介绍
- 响应偏离表

两者严格区分，切勿混淆。
